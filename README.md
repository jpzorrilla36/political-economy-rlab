# Causal Inference in R – Lab

Political Economy I (CSO-15041) — Spring 2026

This repository contains materials for the R lab sessions on causal inference in the social sciences for the Political Economy I course at ITAM. The sessions focus on both the theoretical foundations and practical implementation in R of core causal inference methods used in applied research.

## Topics Covered

### Week 5: Introduction to R and Randomized Control Trials
- R and RStudio setup
- R scripting basics and project organization
- Potential outcomes framework
- Estimation of treatment effects using difference in means and regression
- RCT implementation and inference in R

### Week 5: Instrumental Variables
- Endogeneity and omitted variable bias
- Instrument relevance and exclusion restriction
- Two-Stage Least Squares (2SLS)
- IV estimation in R

### Week 8: Regression Discontinuity Design
- Sharp vs. fuzzy RDD
- Identification assumptions
- Bandwidth selection and robustness
- Graphical analysis
- RDD estimation and visualization in R

### Week 9: Difference-in-Differences
- Parallel trends assumption
- Two-way fixed effects
- Event-study designs
- DiD estimation in R
